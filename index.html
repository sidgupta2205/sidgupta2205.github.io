2. Content Based Image Retrival System - Template Search

Background - This system is part of the ibeyonde Bingo AI system which allow our users to search for an object of interest in any range of dates across mutliple cameras that can be present in different locations. This will provide them the power to search simultaneously in multiple areas without 24/7 monitoring.

Technical Details - Technically this system is a type of content based image retrival system which uses K-nearest neighbour(k-NN) algorithm to determine the images with most similar features. Features are just the textures of an image mapped to a histogram with specified no of bins. The template i.e. the cropped image textures are extracted and plotted in histogram.This histogram is then compared with other feature histogram of images present in the database. The searching is a process of calculating distances(differences) between the histogram. Once the calculation is complete the images are then ranked in increasing order of the distances with the template and finally displayed on the screen. This system utilises the power of artificial intelligence and cloud for smart survellience through your cameras in least possible time and resources.
